Sandbox 
Sandbox is the term used to describe the test system of the training system.  It is a copy of the 
live system where you can learn and practice.   
 
Global Search 
Global search is a powerful tool for users who wish to get faster.  You can search anything in 
NetSuite from this single point.  It takes practice and a desire to learn to master the global 
search field.
