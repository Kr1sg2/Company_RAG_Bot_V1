Button has come completely out of the cushion 
If the button and string have come completely out of the cushion use the supplied button tool to reach back through the 
cushion and reattach the buttons.   
Use Caution, The Button Tool Is Sharp 
Prepare the button 
1. Attach the looped string to one button 
 
Replacing the button 
a. Place the cushion on a flat surface.  Be sure to protect the surface so you do not scratch it with the 
button tool 
b. Locate the existing hole in the fabric where the button was connected.  Slowly insert the button tool 
into the hole.  Do not push the tool all the way through.
