7/14/2020 
 
Setting Up A Conference Call 
The internal Leaders phone system has the ability to facilitate conference calls.  To set up a conference 
call follow these instructions. 
1. Dial 
a. From a company:  Extension 5043  
b. From a phone outside of the company (cell phone):  (727) 285-8907 
2. When prompted enter a pin 
Participant Code 6303 
Host Code 6303126 
There can be more than one host 
3. When the last host leaves the meeting, the conference will end
